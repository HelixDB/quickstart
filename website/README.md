# Website

This is a simple website demo

## Prerequisites

- Node.js installed
- HelixDB running locally on port 6969

## Frontend

```bash
npm install
npm run dev
```

## Running the backend

### Go
```bash
go run go_backend/main.go
```

### Python

```bash
python -m venv venv
source venv/bin/activate
pip install -r requirements.txt
python app.py
```

### Rust

```bash
cargo run
```
